# 🏗️ Архитектура платформы «Weld AI 2.0»

## Обзор системы

Платформа представляет собой мультимодальное решение для контроля качества на производстве подвижного состава с применением **генеративного ИИ** для:

1. Синтеза «идеального» изделия по КД (параметрическая модель → промышленная версия: Diffusion Models)
2. Генерации объяснимых отчётов с причинно-следственным анализом (правило-базированный → промышленная версия: LLM)
3. Контроля технологической последовательности через трекинг объектов

## Компоненты системы

### 1. Слой захвата данных

- **Источники:** IP-камеры, фотоаппараты, архивные фото/видео
- **Форматы:** JPG, PNG, BMP, MP4, AVI
- **Интеграция:** RTSP, ONVIF, локальное хранилище

### 2. Ядро обработки (мультимодальный ИИ)

| Компонент | Технология (демо) | Технология (промышленная) | Назначение |
|-----------|-------------------|---------------------------|------------|
| Детекция дефектов | YOLOv8-seg | YOLOv8-seg + кастомные ансамбли | Обнаружение и сегментация дефектов |
| Синтез «идеального» изделия | Параметрическая модель | Stable Diffusion 2.1 API | Генерация эталонного изображения по КД |
| Объяснимые отчёты | Правило-базированный анализ | TinyLlama 1.1B API | Генерация причинно-следственного анализа |
| Контроль процессов | Трекинг + конечный автомат | ByteTrack + OPC UA интеграция | Валидация последовательности операций |

### 3. Слой интеграции с промышленными системами

- **MES/ERP:** REST API для передачи отчётов о качестве
- **Системы управления оборудованием:** OPC UA для остановки линии при критических дефектах
- **Цифровой двойник:** Интеграция с платформой РЖД

### 4. Пользовательский интерфейс

- **Web-интерфейс:** Streamlit (для демо и пилотных внедрений)
- **Промышленная панель:** Grafana + React (план на 2026 г.)

## Демо-режим

Платформа включает полноценный демо-режим с:

- Синтетической генерацией изображений сварных швов и покраски
- Однокликовым запуском всех сценариев
- Визуализацией генеративного анализа (идеальное изделие + карта отклонений)
- Генерацией объяснимых отчётов без необходимости подключения к внешним API

### Вертикальная оптимизация

- **ONNX Runtime** для ускорения инференса в 3.2×
- **TensorRT** для GPU-оптимизации (цель: 120+ FPS на RTX 4090)
- **Квантизация моделей** (FP16 → INT8) для развёртывания на edge-устройствах

## Соответствие требованиям ТЗ хакатона

✅ **Генеративный ИИ:**

- Демо-версия: параметрическая генерация эталона + правило-базированный анализ
- Промышленная версия: интеграция с Diffusion Models и LLM через API
- Полная архитектура для масштабирования до промышленного применения

✅ **Контроль технологических процессов:**

- Трекинг объектов между кадрами
- Конечный автомат валидации последовательности операций
- Обнаружение нарушений технологической цепочки

✅ **Тестовая среда:**

- Встроенные демо-данные (синтетическая генерация изображений)
- Однокликовый демо-режим для жюри
- Полная документация архитектуры

✅ **Масштабируемость:**

- Микросервисная архитектура
- Поддержка горизонтального масштабирования
- План интеграции с промышленными системами (OPC UA, MQTT)
